# Temporary Utilities

Utility scripts for temporary file management and path operations.

## Purpose

This directory contains helper scripts for temporary file operations and directory management within the .claude/ system. These utilities support library operations that need temporary workspace or path transformations.

## Contents

**update_paths.sh**
- Updates file paths in library references after directory reorganization
- Supports path transformation for refactoring operations
- Used during library structure migrations

## Maintenance

**Retention Policy**: Scripts persist until superseded or removed during cleanup
**Cleanup Schedule**: No automatic cleanup (utility scripts remain available)
