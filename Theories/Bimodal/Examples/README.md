# Examples

Example proofs and demonstration files for TM bimodal logic.

## Contents

| File | Description |
|------|-------------|
| Demo.lean | Quick demonstration of the library |
| BimodalProofs.lean | Combined modal-temporal proofs |
| BimodalProofStrategies.lean | Strategies for bimodal reasoning |
| ModalProofs.lean | Pure modal logic proofs |
| ModalProofStrategies.lean | Strategies for modal reasoning |
| TemporalProofs.lean | Pure temporal logic proofs |
| TemporalProofStrategies.lean | Strategies for temporal reasoning |
| TemporalStructures.lean | Temporal structure examples |

## Purpose

These files demonstrate:
- How to construct derivation trees for common theorems
- Proof strategies for modal and temporal reasoning
- Example uses of automation tactics
