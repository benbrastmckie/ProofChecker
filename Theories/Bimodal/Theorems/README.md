# Theorems

Derived theorems and proof infrastructure for TM bimodal logic.

## Contents

| File | Description |
|------|-------------|
| Combinators.lean | Propositional combinator lemmas (I, K, S, etc.) |
| Propositional.lean | Propositional tautologies and derived rules |
| ModalS4.lean | S4 modal theorems |
| ModalS5.lean | S5 modal theorems |
| GeneralizedNecessitation.lean | Generalized necessitation rules |
| Discreteness.lean | Discreteness-related theorems |
| Perpetuity.lean | Perpetuity principle re-exports |
| Perpetuity/ | Subdirectory for perpetuity theorems |

## Key Categories

- **Combinators**: Identity, composition, permutation lemmas
- **Propositional**: Classical tautologies (double negation, contraposition)
- **Modal S4/S5**: Axiom consequences for S4 and S5 fragments
- **Perpetuity**: P1-P6 perpetuity principles (G-H equivalences)
