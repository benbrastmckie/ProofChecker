# LaTeX Assets

Style files and notation packages for LaTeX documentation.

## Contents

| File | Description |
|------|-------------|
| bimodal-notation.sty | TM bimodal logic notation macros |
| formatting.sty | Document formatting settings |
| notation-standards.sty | Standard notation conventions |

## Usage

Include in LaTeX documents:
```latex
\usepackage{assets/bimodal-notation}
\usepackage{assets/formatting}
```
