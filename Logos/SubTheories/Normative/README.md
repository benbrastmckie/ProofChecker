# Logos.Normative - Layer 3 (Future Extension)

**Status**: Planned for future development

## Overview

The Normative layer extends Core TM logic with operators for deontic reasoning, preferences, and ethical constraints.

## Planned Operators

- **Deontic**: `O` (obligatory), `P` (permissible), `F` (forbidden)
- **Preference**: `≻` (strictly prefers), `≽` (weakly prefers)
- **Ought**: `Ought` (ought to be), `Ought_a` (agent a ought to do)

## Semantic Extension

- **IdealityOrdering**: Extends WorldState with:
  - Ordering over worlds (better/worse)
  - Deontic accessibility (normatively ideal worlds)
  - Preference rankings (agent-specific orderings)

## Timeline

Estimated 3-6 months post Core MVP completion.
