## Overview

This directory contains comprehensive installation guides for ProofChecker (Logos), a formal verification project implementing modal, temporal, and epistemic logics in Lean 4 with Mathlib.

## Quick Navigation

| Guide | Description | Audience |
|-------|-------------|----------|
| [CLAUDE_CODE.md](CLAUDE_CODE.md) | AI-assisted installation with Claude Code | Recommended starting point |
| [BASIC_INSTALLATION.md](BASIC_INSTALLATION.md) | Manual installation (elan, Lean 4, Mathlib) | Command-line users |
| [GETTING_STARTED.md](GETTING_STARTED.md) | Terminal basics, editor setup (VS Code/NeoVim) | New users |
| [USING_GIT.md](USING_GIT.md) | Git/GitHub setup and workflows | Contributors |

## Recommended Reading Order

### For New Users (AI-Assisted)

1. **[CLAUDE_CODE.md](CLAUDE_CODE.md)** - Install Claude Code and let it guide you through setup
2. **[GETTING_STARTED.md](GETTING_STARTED.md)** - Learn terminal basics and set up your editor
3. **[USING_GIT.md](USING_GIT.md)** - Set up Git/GitHub if you plan to contribute

### For Experienced Users (Manual)

1. **[BASIC_INSTALLATION.md](BASIC_INSTALLATION.md)** - Install Lean 4 and build the project
2. **[GETTING_STARTED.md](GETTING_STARTED.md)** - Configure your editor (VS Code or NeoVim)
3. **[USING_GIT.md](USING_GIT.md)** - Set up Git/GitHub for contributions
